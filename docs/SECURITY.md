# Security Policy

**Read also in: [Español], [Português Brasileiro]**

On behalf of security, below is a list of topics on the subject regarding HIGH
and the rest of the code from the Master-Tramador's page on GitHub Pages.

## Cookies and Data Storing

This site does not take, put, or manage any kind cookie currently. Also no data
is stored, neither in local and/or session storage. Thus clear, please note that
in the future it can change due to some settings that will be implemented as, for
example, language.

There is no intent to allow third-party tracking services, and there is no intent
to collect data for commercial or profitable purposes. This site and all its content
will be forever free, of personal divulgation purposes only.

## Sessions

This site does not provide Authentication and/or Authorization services, does not
have any kind of login workflow, and is client-side only. There is no intent to
create a server connection or anything similar.

## Reporting Security Problems

If a security problem is found, please report me directly through any social media,
those found [here], at my profile page. Any Issue will be instantly closed due to
privacy matters regarding security.

If this site or any Mestre-Tramador's content is being hosted on a non Mestre-Tramador
account, please report me too. The links to my social media can be found [here] also.

**All possible future changes regarding security will be noticed and further**
