# Política de Segurança

**Leia também em: [English], [Español]**

Em nome da segurança, segue abaixo uma lista de tópicos sobre o assunto referente
a HIGH e o restante do código da página do Master-Tramador no GitHub Pages.

## Cookies e Armazenamento de Dados

Este site não usa, coloca ou gerencia nenhum tipo de cookie atualmente. Além disso,
nenhum dado é armazenado, nem no armazenamento local e/ou de sessão. Assim esclarecido,
informamos que futuramente poderá sofrer alterações devido a algumas configurações
que serão implementadas como, por exemplo, idioma.

Não há intenção de permitir serviços de rastreamento de terceiros e não há intenção
de coletar dados para fins comerciais ou lucrativos. Este site e todo o seu conteúdo
será gratuito para sempre, apenas para fins de divulgação pessoal.

## Sessões

Este site não fornece serviços de Autenticação e/ou Autorização, não possui nenhum
tipo de fluxo de login e é apenas client-side. Não há intenção de criar uma conexão
de servidor ou algo semelhante.

## Relatando Problemas de Segurança

Se algum problema de segurança for encontrado, por favor denuncie a mim diretamente
através de qualquer rede social, encontradas [aqui], em minha página de perfil.
Qualquer Issue será fechada instantaneamente devido a questões de privacidade
relacionadas à segurança.

Se este site ou qualquer conteúdo de Mestre-Tramador estiver sendo hospedado em
uma conta que não seja de Mestre-Tramador, denuncie a mim também. Os links para
minhas redes sociais também podem ser encontrados [aqui].
