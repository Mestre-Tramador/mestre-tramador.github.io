# Diretrizes de Contribuição

**Leia também em: [English], [Español]**

Em primeiro lugar, muito obrigado desde já, alguém querendo contribuir com um
projeto tão pessoal significa muito para mim. Além disso, antes de scrollar para
baixo, verifique o [Código de Conduta] e a [Política de Segurança], ambos os
arquivos têm informações muito boas se você tiver alguma dúvida.

Abaixo está uma lista simples de contribuições que você pode fazer:

- [Testes]
  - [Relatos de Bugs]
- [Pull Requests]
- [Documentação]
- [Outras Contribuições]

## Testes

Por favor, não tenha medo de clonar meu repositório e testá-lo livremente! Se
você encontrou algum resultado inesperado, por favor, confira abaixo alguns
procedimentos a serem feitos nesses casos...

### Relatos de Bugs

Se você encontrar um bug, por favor, relate-o usando [esta][Relato de Bug] Template
de Issue, descrevendo de forma concisa o comportamento da falha. Além disso, sinto
muito pelo transtorno.

## Pull Requests

Estou estudando neste momento o workflow dos Templates de Pull Requests. Quando
terminar, todos os meus repositórios públicos serão atualizados para tê-los, então,
atualmente, todas as pull requests serão **rejeitadas**.

## Documentação

Uma Wiki será escrita assim que o código tomar forma, logo deve estar disponível.
Se por acaso você encontrar um erro na documentação, verifique
[esta][Revisão de Documentação] Template de Issue para atribuir uma revisão.

## Outras Contribuições

Uma estrela seria recebida com prazer! Além disso, se você tiver alguma dúvida,
faça-a [aqui][Resposta de Pergunta].
