# Politica de Seguridad

**Lea también en: [English], [Português Brasileiro]**

En aras de la seguridad, abajo se muestra una lista de temas sobre HIGH y el resto
del código de la página de Master-Tramador en GitHub Pages.

## Cookies y Almacenamiento de Datos

Este sitio web no toma, coloca ni administra ningún tipo de cookie actualmente.
Tampoco se almacenan datos, ni en el almacenamiento local y/o de sesión. Así claro,
tenga en cuenta que en el futuro puede cambiar debido a algunas configuraciones
que se implementarán como, por ejemplo, el idioma.

No hay intención de permitir servicios de seguimiento de terceros, y no hay intención
de recopilar datos con fines comerciales o lucrativos. Este sitio web y todo su
contenido será gratuito para siempre, únicamente con fines de divulgación personal.

## Sesiones

Este sitio web no proporciona servicios de Autenticación y/o Autorización, no tiene
ningún tipo de flujo de login y es solo client-side. No hay intención de crear una
conexión de servidor o algo similar.

## Informando Problemas de Seguridad

Si se encuentra un problema de seguridad, infórmeme directamente a través de
cualquier red social de las que se encuentran [aquí], en mi página de perfil.
Cualquier Issue se cerrará instantáneamente debido a cuestiones de privacidad
relacionadas con la seguridad.

Si este sitio web o cualquier contenido de Mestre-Tramador está alojado en una
cuenta que no es de Mestre-Tramador, infórmeme también. Los enlaces a mis redes
sociales también se pueden encontrar [aquí].
