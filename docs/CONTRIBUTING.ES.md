# Pautas de Contribución

**Lea también en: [English], [Português Brasileiro]**

En primer lugar, muchas gracias de antemano, alguien que quiera contribuir con
un proyecto tan personal significa mucho para mí. Además, antes de desplazarse
hacia abajo, consulte el [Código de Conducta] y la [Política de Seguridad], ambos
archivos tienen muy buena información si tiene alguna duda.

Abajo se muestra una lista simple de las contribuciones que puede hacer:

- [Pruebas]
  - [Informes de Bugs]
- [Pull Requests]
- [Documentación]
- [Otras Contribuciones]

## Pruebas

¡Por favor, no tenga miedo de clonar mi repositorio y probarlo libremente! Si
encuentra algún resultado inesperado, por favor, consulte abajo algunos procedimientos
a seguir en estos casos...

### Informes de Bugs

Si encuentra un error, infórmelo usando [esta][Informe de Bug] Plantilla de Issue,
y describe de manera concisa el comportamiento de la falla. Además, lamento mucho
por el problema.

## Pull Requests

Estoy estudiando en este momento el flujo de trabajo de las Plantillas de Pull Requests.
Cuando termine, todos mis repositorios públicos se actualizarán para tenerlos,
por lo que actualmente todas las pull requests serán **rechazadas**.

## Documentación

Se escribirá una Wiki así que el código tome forma, entoces por lo que pronto deberá
estar disponible. Si por casualidad encuentra un error en la documentación, consulte
[esta][Revisión de Documentación] Plantilla de Issue para asignar una revisión.

## Otras Contribuciones

¡Una estrella sería recibida con mucho gusto! Además, si tienes alguna pregunta,
hazla [aquí][Respuesta de Pregunta].
